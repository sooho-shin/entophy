# Entropy Explorer

Entropy Explorer project

# 실행방법

1. Install YARN packages
   ```sh
   yarn
   ```
2. Start Dev
   ```sh
   yarn dev
   ```
3. Build
   ```sh
   yarn build
   ```
   빌드시 자동 배포
